# 개발기간 및 인원
- 개발 기간 : 2022년 5월 9일 ~ 2022년 5월 20일 (12일)
- 개발 인원 : 프론트엔드 (박준범(PM), 이나연, 이형준, 김병규), 백엔드 (김영빈, 제정욱)

# 사용 기술 및 협업툴
- Back-End : Python, Django, AWS, JWT, bcrypt, MySQL, CORS headers, RESTful API
- 협업 툴 : Git, Trello, Slack, Notion, Google Spreadsheet
